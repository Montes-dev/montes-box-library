# Icons Directory

Place icon assets for the MB Library here, such as:
- UI icons
- Status indicators
- Action icons
- Menu icons
- Tool icons

## Recommended Formats
- PNG (with transparency)
- SVG (vector-based for scaling)

## Recommended Sizes
- Small: 16x16, 24x24
- Medium: 32x32, 48x48
- Large: 64x64, 128x128
